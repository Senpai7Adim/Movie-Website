## Introduction

Welcome to the Movie Website project! This beginner-friendly site is built with **HTML**, **CSS**, and **JavaScript**. It features a modern, fully responsive UI, smooth reload animation, an interactive navigation bar, and an animated carousel slider. Explore trailers, download options, and more.

## Features

- Responsive movie/series listing site
- Carousel slider with images and videos
- Download and trailer viewing buttons
- Stylish navigation and search box
- Modern animations with CSS & JS

## Getting Started

1. Clone this repository
   ```
   git clone https://github.com/Senpai7Adim/Movie-Website.git
   ```
2. Open `index.html` in your browser
3. No build tools required!

## Technologies Used

- HTML5
- CSS3 (responsive design and animations)
- JavaScript

## Project Structure

- `index.html` - Main website page
- `style.css` - All styling and responsiveness
- `app.js` - JavaScript for UI interactions and slider
- `/image/` - All project images and thumbnails
- `/video/` - Series trailers and video files
